# Supabase

هذا المجلد يحتوي ملفات قاعدة البيانات والتخزين للموقع.

- `schema.sql`: الجداول، القيود، الفهارس، الدوال، Triggers، وRLS.
- `seed.sql`: بيانات بداية قابلة للتعديل مثل الباقات والأسئلة.
- `storage-policies.sql`: Buckets وسياسات رفع الصور للإدارة فقط.

ملاحظات مهمة:

- لا تستخدم Service Role Key داخل الموقع.
- الواجهة تستخدم Supabase URL وAnon Public Key فقط.
- الزائر يستطيع إضافة بيانات في نماذج الانتظار والتواصل والدعم وحذف الحساب فقط.
- قراءة وتعديل بيانات الإدارة محمية بـ Supabase Auth وRLS.
- يمكن إضافة Cloudflare Turnstile لاحقًا لتقليل Spam بجانب Honeypot الموجود في النماذج.
