# Testing Checklist

استخدم هذه القائمة قبل اعتبار الموقع جاهزًا للنشر:

- [ ] فتح كل الصفحات العامة.
- [ ] التأكد أن كل روابط التنقل تعمل.
- [ ] اختبار قائمة الموبايل.
- [ ] اختبار زر تبديل اللغة عربي / English.
- [ ] اختبار نماذج Waitlist وContact وSupport وDelete Account بدون Supabase ورسائلها واضحة.
- [ ] اختبار نفس النماذج بعد ربط Supabase.
- [ ] اختبار تسجيل دخول الإدارة.
- [ ] التأكد أن صفحات الإدارة لا تفتح بدون مستخدم إداري.
- [ ] اختبار إضافة وتعديل وحذف خبر.
- [ ] اختبار رفع صورة خبر إلى Storage.
- [ ] اختبار صفحة الأخبار وتفاصيل الخبر.
- [ ] اختبار لوحة الرسائل وتغيير الحالة.
- [ ] اختبار قائمة الانتظار والتصدير CSV.
- [ ] مراجعة صفحة 404.
- [ ] مراجعة التصميم على موبايل وتابلت وديسكتوب.
- [ ] التأكد من عدم وجود Broken images أو ملفات مفقودة.
- [ ] التأكد من عدم وجود Secrets داخل الملفات.
- [ ] مراجعة RLS: الزائر لا يستطيع قراءة جداول الإدارة.
- [ ] مراجعة `_headers` وسياسة Content Security Policy.
- [ ] تحديث `sitemap.xml` وCanonical URLs بالدومين الحقيقي قبل النشر.
